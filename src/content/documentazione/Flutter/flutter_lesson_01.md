# Introduzione a Flutter

## Cos’è Flutter

Flutter è un framework per creare applicazioni **cross-platform**.

Si basa sul linguaggio **Dart** → è necessaria una base di Dart prima di iniziare.

Con un’unica base di codice si possono sviluppare app per:


- Mobile (focus del corso)
- Web
- Desktop


---


## Compilazione e Performance

Il codice scritto in Flutter viene compilato in **codice nativo**.

Non utilizza un semplice “contenitore” come altri framework cross-platform.

Le performance sono simili a:

- App Android scritte in `Kotlin`
- App iOS scritte in `Swift`

👉 **Vantaggio principale:** un solo linguaggio invece di doverne imparare diversi per web, mobile e desktop.

---

## 🔥 Hot Reload

Funzionalità chiave: **Hot Reload**.

Permette di vedere le modifiche quasi istantaneamente.

- Riduce drasticamente i tempi di attesa
- Migliora enormemente la produttività

## 🧱 Concetto fondamentale: i Widget

In Flutter **tutto è un Widget**.

Un’app è composta da tanti piccoli componenti:

- Testo
- Icone
- Card
- Immagini
- Layout

Grande attenzione ai:

- Widget di layout
- Gestione dello stato
- Login e registrazione
- Notifiche
- Geolocalizzazione
- Interazione con API
